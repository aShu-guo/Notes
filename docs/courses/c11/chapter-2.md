# 分支结构

## 运算符

### 算数运算符

- \+
- \-
- \*
- \/
- \%

乘除的优先级最高，模运算的优先级与除相等，最后是加减

### 逻辑运算符

- &&：短路与 对于连接多个`&&`的表达式，当前面表达值`为0`时，就`不再继续执行`
- ||：短路或 对于连接多个`||`的表达式，当前面表达式值`非0`时，就`不再继续执行`
- !：非

### 三目运算符

- 一目运算符：参与运算的有1个量，例如++、--
- 双目运算符：参与运算的有2个量，例如+、-、*、/
- 三目运算符：参与运算的有3个量，例如：?...:

s=1 k=4
s=2 k=4
